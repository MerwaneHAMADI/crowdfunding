# Crowdfunding platform

This project is an onboarding training in Capsens

## Getting Started

### Prerequisites

* **Ruby 2.6.3** (might work with other versions but not guaranteed)
* **Rails 2.6.3**


### Installing

```
git clone git@github.com:MerwaneHAMADI/crowdfunding.git
yarn install
rails db:migrate
rails s
```
